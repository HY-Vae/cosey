# TransitionGroup 过渡组

## 简介

`TransitionGroup` 是对 `vue transition-group` 的简单封装，预设了一些动画效果。

### 基础演示

## API

### TransitionGroupProps

| 属性   | 描述     | 类型                        | 默认值 |
| ------ | -------- | --------------------------- | ------ |
| effect | 动画效果 | 'slide' \| 'flip' \| 'fade' | 'fade' |

### TransitionGroupSlots

| 插槽    | 描述         | 属性 |
| ------- | ------------ | ---- |
| default | 定义默认内容 | -    |
